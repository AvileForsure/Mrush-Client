# Mrush.pl Client

A Simple Client For Pogromcy/Mrush.pl

## Run Without Building

Clone the project

```bash
  git clone https://github.com/AvileForsure/Mrush-Client
```

Go to the project directory

```bash
  cd Mrush-Client
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run start
```

## Build App

Clone the project

```bash
  git clone https://github.com/AvileForsure/Mrush-Client
```

Go to the project directory

```bash
  cd Mrush-Client
```

